The files are:
  - `bruno_v5.py` : You play manually by entering moves in the terminal. Depth = 4
  - `run_match.py` : You make Bruno against Stockfish. For that you need to have `stockfish.exe` in you local directory.
  - `bruno_chess.py` : Running this will allow you play against `bruno_v5.py` but in a GUI.
  - `bruno_chess.html` : The web-based GUI to play chess against Bruno

You are free to make updates and let me know!
